## hawkeye

Hawk-Eye, the infrastructure monitoring agent

### Synopsis

Hawk-Eye is an infrastructure monitoring agent that is able to perform different checks.
The check results are exposed via an API.

### Options

```
  -c, --config string   config file (default is $HOME/.hawkeye.yaml)
  -h, --help            help for hawkeye
```
