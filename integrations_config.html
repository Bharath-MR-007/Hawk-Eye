HPE NNMi Integration

Connect Hawk-Eye to HPE Network Node Manager i (NNMi) to forward incidents as NNMi management events. Hawk-Eye uses the NNMi Web Services REST API (nnm/rest/) to create incidents when alerts fire.
The fully-qualified hostname or IP address of the NNMi management server.
Default: 9443 (HTTPS) or 80 (HTTP). Check NNMi admin console for your configured port.
The NNMi account must have the Web Services Client role assigned in NNMi → Security → User Accounts.
Password for the NNMi Web Services user account.

e.g. mixtral, llama3, gpt-4o, claude-3-opus
